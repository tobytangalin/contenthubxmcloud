Limitations and restrictions

Abstract

Describes some limitations and restrictions when you're using XM Cloud.

There are many components in XM Cloud that will be familiar to developers who have worked with Sitecore Experience Manager and Sitecore Experience Platform. The following features, modules, and roles are available in Sitecore Experience Manager or Sitecore Experience Platform, but not in XM Cloud.

Features and modules

The following features and modules are not in XM Cloud:

  • XP/xConnect/xDB functionality including existing personalization rules (a new embedded personalization experience is available for XM Cloud)

  • Marketing applications, including Experience Analytics, Experience Profile, FXM, Experience Optimization, List Manager, Campaign Creator, Path Analyzer, Marketing Automation and the Marketing Control Panel

  • Email Experience Manager (EXM)

  • SXA with MVC, and SXA Search components

  • Sitecore Forms

  • Custom search indexes

  • Identity server

  • Sitecore Publishing Service

  • Universal Tracker

  • Device detection

  • IP Geolocation

  • MVC

  • The Update Center

Roles

XM Cloud does not contain any of the roles related to XP/xConnect functionality, such as the Marketing Automation Engine or xDB Processing. In addition, the following roles found in Sitecore Experience Manager are not in XM Cloud:

  • Content Delivery

  • Content Publishing

  • Forms database

  • Web database

  • Web index

  • Sitecore Identity Server

  • Device Detection

  • IP Geolocation
