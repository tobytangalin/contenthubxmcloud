Sitecore editors and content management

Abstract

Overview of the JSS support for integrating advanced Sitecore content and layout editors

Applications created with the Sitecore JavaScript Rendering SDK (JSS) can be integrated with advanced Sitecore editors such as the Experience Editor and Pages.

Editor support for JSS Next.js applications

Applications created with JSS for Next.js support the following advanced Sitecore editors:

  • Experience Editor

  • Pages

For an overview of editor integration, refer to Architecture and APIs for integrating JSS Next.js apps with Sitecore editors.

Editor integration is configured by default in JSS Next.js application included in the XM Cloud Next.js Starter Kit containers.

For applications you create with the JSS initializer, you must configure the integration.
