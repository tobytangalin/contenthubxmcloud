Sites

The SXA content architecture includes site collections and sites. SXA supports multiple site collections, which means that you can run multiple sites on a single instance of Sitecore. Each site collection can include multiple related sites, for example, to support multiple brands for a single company or multiple languages or locations for a single brand.
