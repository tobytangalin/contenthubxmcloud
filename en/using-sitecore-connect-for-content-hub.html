Using Sitecore Connect for Content Hub

Abstract

Introduction to tips and walkthroughs for using the Sitecore Connect for Content Hub connector

This section contains procedures that explain how to customize and use the Sitecore Connect for Content Hub connector. The procedures show you how to, for example, create a trigger in the Sitecore Content Hub, add a field type converter, or change the Image tag attribute from DAM.
